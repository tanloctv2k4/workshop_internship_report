This section provides the source code package and demo video of the **PharmaCare AI** project after the system was deployed on AWS.

The source code and video demo are stored on Google Drive for convenient access and review.

---

## 1. Source Code

The source code includes the main components of the **PharmaCare AI** system:

* Frontend source code using ReactJS.
* Backend API source code using AWS Lambda with Node.js 22.x.
* Database migration scripts for Amazon RDS PostgreSQL.
* AI/RAG source code for Lambda Indexing and Lambda Chatbot.
* Configuration files used during development and deployment.

> Note: Sensitive information such as AWS credentials, database password, secret values, and environment-specific keys are not included in the shared source code.

## 2. Demo Video

The demo video presents the main features of the **PharmaCare AI** system, including:

* Accessing the deployed website through CloudFront.
* Viewing product categories and product details.
* User authentication using Amazon Cognito.
* Customer functions such as cart and order creation.
* Admin functions such as product, user, and order management.
* AI chatbot consultation using the RAG architecture.
* Monitoring, backup, and security configuration overview.
